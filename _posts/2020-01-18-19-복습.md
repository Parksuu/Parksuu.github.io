2. 10진수 n을 m진수로 만들기

   몫이 m보다 작을때까지 나누기 **&** 기저에서부터 출력하는지 아닌지 확인해주기

   ```java
   	static public void sol(int n , int m) {
   		//숫자 n과 m진수
   		if(n<m) {
   			System.out.print(n);
   			return;
   		}
   		else {
   			sol(n/m, m);
   			System.out.print(n%m);
   		}
   	}
   ```
